# 📚 dudamelhorado

## 📌 Descrição
O **dudamelhorado** é um algoritmo escrito em **Portugol** para controle de acesso em instituições de ensino.  
Ele organiza a entrada de **alunos** e **professores**, garantindo que apenas pessoas cadastradas possam entrar na sala de aula ou faculdade.

## 🎯 Objetivos
- **[Verificação](ca://s?q=Verificação_em_pseudocodigo)**: checar se o nome está presente na lista oficial.  
- **[Condicional Positiva](ca://s?q=Condicional_Positiva_em_pseudocodigo)**: permitir a entrada se o nome estiver na lista.  
- **[Condicional Negativa](ca://s?q=Condicional_Negativa_em_pseudocodigo)**: negar a entrada e exibir mensagem de erro se não estiver cadastrado.  
- **[Repetição](ca://s?q=Laço_de_repetição_em_pseudocodigo)**: repetir o processo até que todos da fila sejam verificados.  

## 🛠️ Estrutura Lógica
O programa utiliza:
- Estrutura **Se-Então-Senão** para decisões.  
- Estrutura **Para** para percorrer todos os nomes da fila.  
- Mensagens claras de saída para informar o resultado da verificação.
