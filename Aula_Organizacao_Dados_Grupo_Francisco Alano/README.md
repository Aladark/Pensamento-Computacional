# ✈️ Projeto - Organização de Dados: Rotas de Voo

## 📌 Tema
O conjunto escolhido foi **Rotas de Voo / Tráfego Aéreo**.

## 🎯 Objetivo
Representar o mesmo conjunto de informações em três estruturas computacionais distintas:
- **Fila (Queue)** → sequência ordenada de voos.
- **Grafo (Graph)** → conexões entre aeroportos e rotas aéreas.
- **Hierarquia (Hierarchy)** → níveis de organização do tráfego aéreo.

## 📂 Estrutura do Repositório
Aula_Organizacao_Dados/
│── README.md
│── Fila.png
│── Grafo.png
│── Hierarquia.png


## 📖 Descrição das Estruturas
- **Fila:** Representa os voos em ordem de decolagem, como uma sequência linear.  
- **Grafo:** Mostra como diferentes aeroportos estão conectados por rotas aéreas.  
- **Hierarquia:** Estrutura organizacional do tráfego aéreo, do hub principal até destinos regionais.
