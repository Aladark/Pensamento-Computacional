# 🎓 Projeto Acadêmico - Flask

## 📌 Objetivo
Este projeto tem como finalidade implementar um **sistema acadêmico** em Flask, com autenticação, gerenciamento de disciplinas, notas, recomendações e relatórios.  
A atividade consiste em identificar erros, corrigi-los e avaliar a solução final quanto à clareza, eficiência e escalabilidade.

## 📂 Entregáveis
- `README.md` → explicação da atividade e objetivos.
- `Erros_Identificados.md` → registro dos erros encontrados.
- `Projeto_Corrigido.md` → versão corrigida com justificativas.
- `Avaliacao.md` → reflexão sobre eficiência e clareza da solução final.
